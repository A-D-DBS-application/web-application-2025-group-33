## Authentication

As a User, I want to register as an Author or a Company, so that I can access the features relevant to my role.

As a User, I want to log in, so that I can reach my personal dashboard.

## Author Stories

### Create & Manage Papers

As an Author, I want to create a new paper with a title, subject and PDF upload, so that I can start publishing my research.

As an Author, I want to update my paper's title, subject or PDF version, so that the information stays accurate over time.

As an Author, I want to publish my paper, so that companies can discover it, mark it as "interested" and download it.

As an Author, I want to delete my paper, so that I maintain control over outdated or unwanted content.

As an Author, I want to see a list of my papers, so that I can manage them efficiently.

### Collaboration

As an Author, I want to add other authors as collaborators, so that we can work together on the same paper.

As an Author, I want to remove collaborators, so that I can manage who has editing permissions.

### Insights

As an Author, I want to see download counts and interest statistics, so that I understand how my paper performs.

As an Author, I want to view company interest in my paper, so that I know who finds my work relevant.

As an Author, I want to read reviews from companies, so that I can learn how my research is perceived.

## Company Stories

### Discover Papers

As a Company, I want to browse published papers, so that I can discover relevant research.

As a Company, I want to search papers by title, author or field, so that I can quickly find what I'm looking for.

As a Company, I want to view detailed paper pages, so that I can assess a paper's relevance.

As a Company, I want to download PDFs, so that I can review papers in depth.

### Interests

As a Company, I want to mark a paper as "interested," so that I can keep track of research that matters to me.

As a Company, I want to remove interest, so that I can refine my personal collection.

As a Company, I want to view all papers I marked as interested, so that I can easily revisit them.

### Recommendations

As a Company, I want to set my research interests, so that the platform can recommend relevant papers.

As a Company, I want to receive recommended papers, so that I quickly find work aligned with my needs.

## Reviews

As a Company, I want to leave a rating and comment on a paper, so that I can share feedback with the authors.

As an Author, I want to be prevented from reviewing my own paper, so that scoring stays fair.

As a User, I want to read existing reviews, so that I understand how others evaluate a paper.
